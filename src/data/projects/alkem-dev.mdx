The Alkemical Development website itself is open source. It serves as both
our public face and a reference implementation for our preferred web stack.

## Stack

- **Astro 5** — islands architecture, zero JS by default
- **Tailwind CSS 4** — CSS-only config, fast builds
- **Three.js** — custom GLSL shaders for the landing page hero
- **OpenTofu** — infrastructure as code for Cloudflare Pages
